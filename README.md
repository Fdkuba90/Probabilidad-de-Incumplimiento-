# PI Empresarial - FINANTAH

App web para cargar PDFs del Buró de Crédito Empresarial, calcular el puntaje CNBV por ID, estimar la Probabilidad de Incumplimiento (PI) y generar reportes PDF con marca institucional.

Funcionalidad:
- Subida de PDF
- Extracción automatizada de la sección "Califica"
- Cálculo de puntos por ID
- Cálculo de PI
- Visualización de resultados
- Generación de PDF con logo de FINANTAH, nombre de empresa y fecha
